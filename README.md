# Calculator for Android and IOS
I have made a Calculator app for mobile phones.
## It includes these 
1) Standard or Metric Calculator
2) Scientific Calculator
3) Graphing Calculator
4) Conversions
5) BMI-Calculator
6) Length Converter

I will include more features like Temperature Conversions, Currency Conversions, Area, Volume , etc.
